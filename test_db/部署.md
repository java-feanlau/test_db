# 如何启动

```ini
cd data

mkdir -p ./data ./logs ./conf


docker build -t mysql .

# $PWD表示当前路径


docker run -p 3306:3306 --name mymysql -v  $PWD/data/conf:/etc/mysql/ -v $PWD/data/logs:/logs -v  $PWD/data/data:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=password -d mysql:5.7

# 删除所有未运行的容器
sudo docker rm $(docker ps -a -q)

# 删除所有镜像，-f 可以强制删除
sudo docker rmi $(docker images -q)

```

## MacOS本地上连接

自己本地电脑上的docker mysql容器，用的命令如下：

```
> mysql -u root -p -h 0.0.0.0
```

> 记住一点，不需要加端口号的
